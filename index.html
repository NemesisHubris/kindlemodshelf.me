Kindle Modding Tools & Resources

The most essential jailbroken Kindle tools and plugins. Everything here requires a jailbroken Kindle unless stated otherwise.

Start here:
  • Kindle Modding Wiki: kindlemodding.org
  • Jailbreak required: Use the WinterBreak guide.
Most Popular & Essential Tools
KOReader
Essential
Powerful open-source document reader for Kindle supporting EPUB, PDF, DjVu, CBZ/CBR, and more. Adds features like custom screensavers, USB/network transfer, Bluetooth audio, plugins, dictionary, theming, and advanced file management.
KindleFetch
Essential
Command-line tool to download books from Anna’s Archive directly to your Kindle using the built-in terminal. Requires KTerm terminal.
KTerm (Kindle Terminal Emulator)
Essential
Terminal emulator for Kindle. Access a command-line interface for running scripts, mods, and tools directly on your device.
KOReader Plugins
Z-Library Plugin for KOReader
Plugin
Search and download eBooks from Z-Library directly in KOReader.
Send Highlights & Screenshots via Telegram
Plugin
Send your Kindle highlights and screenshots instantly to your Telegram chat using this plugin and bot.
Weather Plugin
Plugin
Get real-time weather updates on your KOReader home screen.
Clock Plugin (dtDisplay)
Plugin
Display time and date in a fullscreen widget inside KOReader.
Games
Tetris for kterm (HF only)
Game
Classic Tetris game playable in the Kindle's terminal on hackable firmware models. Requires KTerm terminal.
Gnome Games (Chess, Mines)
Game
Play classic Gnome Chess and Minesweeper games on your Kindle.
Nine Men's Morris (HF)
Game
Classic Nine Men's Morris board game for Kindle HF devices. Requires KTerm terminal.
KWordle
Game
Wordle-inspired word guessing game ported to Kindle.
IllusionChess
Game
Chess implementation built on the Illusion framework for Kindle.
Universal Paperclips (HF Port)
Game
Idle clicker game where you turn the world into paperclips. Ported for hackable firmware Kindles.
Game Boy Emulators
Emulator
Play Game Boy and Game Boy Color games on your Kindle using either Gambatte-K2 or Gambatte-K emulators.
Media & Audio
SOX Media Player (HF, Beta)
MediaAudio
Play WAV audio files over Bluetooth using SOX player. Still in beta — not all features are stable, improvements ongoing.
Other Useful Tools
KindleKraft (Minecraft Server)
Tool
Turns your Kindle into a Minecraft LAN multiplayer server. SF-only (unless Java replaced manually)
Disable Ads %
Tool
Removes ads and enables custom KOReader screensavers on ad-supported Kindles. Required to use custom screensavers with KOReader if your Kindle has ads.
Shortcut Browser
Tool
Browse any website fullscreen on your Kindle using this simple web browser.
KAnki (Flashcard Scriptlet)
Tool
Spaced repetition flashcards built on the Illusion framework; runs directly on Kindle without KOReader.
Check OTA Status
Tool
Quickly verify and block Kindle OTA update status to keep your jailbreak safe.
Alpine Kindle (New & Old)
ToolLegacy
Alpine Kindle modding toolkit for older devices. New community-maintained releases and the original repository included.
KindleMod MobileRead Thread (Legacy)
ToolLegacy
Discussion of legacy KindleMod tools. Not confirmed working.
Development
kindle-usbnetlite (SSH via USB)
Dev
Lightweight USB networking utility for Kindle. Lets you SSH into your Kindle from your computer over USB for modding and troubleshooting.
Illusion Framework
Dev
Generic framework enabling scriptlets and apps on Kindle. Use it as the backbone for games and utilities like KAnki and IllusionChess.
ScreenControl
Dev
Control and test Kindle screen refresh, brightness, and patterns—handy for development and fun demonstrations.
Telnet (Upgraded)
Dev
Lightweight Telnet server script for remote CLI access—a legacy alternative to SSH useful for quick debugging.
KPM (Kindle Package Manager)
Dev
Package manager for Kindle providing easy installation and management of mods and utilities.
