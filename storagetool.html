StorageTool – Kindle Storage Manager

StorageTool is a storage management utility for Kindle e-readers. Analyze, visualize, and clean up your device’s storage with detailed overviews, duplicate detection, and fast cleanup tools—all directly from KTerm.

What Is StorageTool?

StorageTool lets you instantly see how your Kindle’s storage is being used, identify large or duplicate files, and organize your books. Ideal for large collections or when your Kindle is running low on space.

Features

  • Storage overview: View total, used, and free space at a glance.
  • Book directory scan: Analyze file sizes and types in your documents folder.
  • Format breakdown: See storage usage by file type (MOBI, PDF, AZW3, etc).
  • Recent files: Find newly added or modified books.
  • Duplicate finder: Locate and review potential duplicate files.
  • Settings: Set main directory, show hidden files, enable debug mode, and more.
  • Auto-updater: Download and install the latest StorageTool version from within the app.

Requirements

  • KTerm terminal emulator

How to Install

  1. Open KTerm on your Kindle.
  2. Run this command to install automatically:
    curl https://jpt.bio/StorageTool/install.sh | sh
  3. Or, for manual install: Download StorageTool from the official GitHub repository.
  4. Copy the storagetool folder to your Kindle’s /mnt/us/extensions/ directory.
  5. Set executable permissions for the scripts:
    chmod +x /mnt/us/extensions/storagetool/run.sh
    chmod +x /mnt/us/extensions/storagetool/bin/storagetool.sh

How to Use

  1. Open KTerm.
  2. Run StorageTool with:
    sh /mnt/us/extensions/storagetool/run.sh
  3. Use the menu to view storage usage, scan for large files, find recent additions, or locate duplicates.
  4. Adjust settings to choose your directory, show hidden files, and customize your experience.

Troubleshooting

  • Tool not running? Make sure you have KTerm installed and that all scripts have executable permissions.
  • Still having issues? Reinstall StorageTool or review the official GitHub instructions.
