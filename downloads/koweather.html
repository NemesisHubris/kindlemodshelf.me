KOWeather Plugin for KOReader

KOWeather is a simple weather forecast plugin for KOReader.
It brings live weather info—right to your Kindle or other e-ink device—so you can check the forecast before your next reading session outdoors.
Main Features
  • Current weather conditions: See what’s happening outside right now
  • 3-day forecast: View today’s forecast and the next two days
  • Hourly forecast: Check the forecast by hour for the current day
  • Choose your location: Set your forecast location by postal code
  • Celsius & Fahrenheit: (Switching units is in progress)
  • 24-hour or 12-hour clock: Customize the display to your preference
How to Install
  1. Download the latest release: KOWeather Plugin on GitHub
  2. Unzip the downloaded file on your computer.
  3. Copy the weather.koplugin folder to your KOReader device:
    Place it in /mnt/us/.adds/koreader/plugins/ (Kindle or Kobo).
    (If that folder doesn’t exist, create it.)
  4. Restart KOReader if it’s running.
  5. Open KOReader and find Weather in your Plugins menu.
How to Use
  1. Open the Weather plugin in KOReader.
  2. Enter your postal code to set your forecast location.
  3. See current conditions, forecasts, and hourly details—directly on your e-ink screen.
Compatibility
  • Tested on: Kobo Libra H2O (may work on other e-ink devices with KOReader)
  • Kindle: Try it! If it works (or doesn’t), let the developer know by opening an issue on GitHub.
Notes
  • Weather data is provided by WeatherAPI.
  • This is a basic, lo-fi weather tool made for e-readers—perfect for quick checks before you read outside.
With KOWeather, you can always know if it’s the right moment to grab your Kindle and enjoy a book under the open sky.
