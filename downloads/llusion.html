Illusion: Make Apps With HTML, CSS, JS on Kindle

Illusion lets you create simple HTML, CSS, and JavaScript apps on your Kindle, using the built-in Mesquite web app loader (the same technology powering the Kindle Store and Browser). Make your own web-based Kindle “apps” and run them natively on your e-reader.
What Is Illusion?
Illusion is a framework for making and running web-based apps on your Kindle. You can build custom tools, games, and utilities using classic web technologies—HTML, CSS, and JavaScript (ES5). Mesquite powers the rendering, so your app runs as a real Kindle application.

Apps can be persistent, keep their own data, and act much like official Kindle web apps.
Key Features
  • Create and run your own HTML/CSS/JS apps on Kindle (via Mesquite)
  • Persistent environment—apps can store state and data
  • Supports most ES5 JavaScript and older CSS standards (no flexbox, no modern JS features)
  • Easy to adapt existing apps—just add a config.xml (see example below)
  • Share, modify, and experiment with your own Kindle web apps
How It Works
Build your web app using HTML, CSS, and ES5 JavaScript. Add a config.xml file (required for Mesquite compatibility), and use the Illusion scriptlet to launch it on your Kindle.

Note: Mesquite uses an old version of WebKit, so you must use ES5 JavaScript and older CSS (avoid flexbox, grid, and ES6+ syntax). Use Babel to transpile newer JS if necessary.
Compatibility
Illusion apps work on any Kindle supporting Mesquite (all modern jailbroken Kindles). Designed for experimentation and hobby development.
Developer Notes
  • Always test on real Kindle hardware—browser emulators may behave differently.
  • Polyfill unsupported features and avoid modern web APIs.
  • Upcoming versions may allow more advanced system integration for powerful utilities (e.g., running shell commands from within apps).

For a practical example, see: IllusionChess on GitHub.

Illusion opens the door to Kindle homebrew development—build and run your own web apps right on your e-reader.
