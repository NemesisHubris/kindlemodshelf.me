Send Highlights & Screenshots via Telegram (KOReader Plugin)

This KOReader plugin lets you quickly send your Kindle book highlights and screenshots directly to your Telegram account using the @bookshotsbot Telegram bot.
No more copying text by hand—just highlight and send!
Features
  • Send book highlights: Instantly share selected text or existing highlights from KOReader to your Telegram chat.
  • Send screenshots: Capture and send Kindle screenshots to Telegram in seconds.
  • Bulk upload bookmarks: Send all your bookmarks from a book at once, or just individual ones.
  • Auto Wi-Fi: Automatically turns on Wi-Fi to send, then turns it off again (where supported).
  • Quote customization: Use the Telegram miniapp to style your quote images with backgrounds, colors, and Unsplash photos.
How to Install
  1. Download the latest release: Plugin on GitHub
  2. Unzip the downloaded file. You’ll get a folder named telegramhighlights.koplugin.
  3. Copy the folder to your KOReader’s plugins directory.
  4. Restart KOReader. The plugin appears under Tools > Telegram Highlights.
How to Set Up
  1. Get a code: Start a chat with @bookshotsbot on Telegram to receive your verification code.
  2. Enter your code in KOReader:
    Open KOReader → Tools → Telegram Highlights → Set verification code → Enter and save the code.
How to Use
  • Send a new highlight: Select text, open the highlight menu, choose Send to Bot or Save & Send.
  • Send an existing highlight: Tap a highlight, press the menu (three dots), select Send to Bot or Save & Send.
  • Bulk send bookmarks: Go to Bookmarks, tap the menu icon, and choose Send all to Bot.
  • Send a single bookmark: Long-press a bookmark, then tap Send to Bot.
  • Send a screenshot: Swipe diagonally on your Kindle, then tap Send to Bot in the dialog.
Customization
Use the Telegram miniapp to change quote image backgrounds, colors, or use Unsplash images.
Launch the miniapp from the Highlights screen in KOReader.
Requirements
  • KOReader installed on your e-reader
  • Telegram account
  • Wi-Fi for sending highlights/screenshots
