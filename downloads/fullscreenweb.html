Fullscreen Web Browser for Kindle (Firmware 5.16.4+)

Fullscreen Web Browser is a custom browser for jailbroken Kindles running firmware 5.16.4 and above. It launches any website in true full screen, perfect for reading, reference, or web apps right on your e-reader.
Features
  • Open any website in a clean, fullscreen view
  • Easily change the homepage to any site you want
  • Minimal interface for maximum reading space
  • Script included to stop the browser if needed
  • Designed for Kindle firmware 5.16.4 and newer
How to Install & Use
  1. Download the latest ZIP:
    Get it from GitHub Releases
  2. Extract the contents:
    Unzip and copy everything to the / (root) directory of your Kindle.
  3. Set your website:
    • Method 1: Edit the index.html in the fullscreenbrowser folder. Replace https://example.com with your desired site.
    • Method 2: Open documents/shortcut_browser.sh and set the FULLSCREEN_SITE variable to your preferred URL.
  4. Launch the app:
    Open the shortcut script from your Kindle’s library to launch the fullscreen browser.
    (Tip: You can rename the script for multiple website shortcuts!)
Stopping the Browser
If the browser process becomes unresponsive, simply run the included shortcut_stop.sh from your Kindle library, or restart your device.
Options
  • GO_FULLSCREEN: Set this to false in the script if you want to keep the Kindle’s UI visible instead of true full screen.
Known Issues
  • Some websites may not load in the index.html method (iframe limitation). For best results, use the FULLSCREEN_SITE variable in the script.
  • If the browser freezes or acts up, use the Stop Shortcut Browser script or restart your Kindle.
Summary
With Fullscreen Web Browser, your Kindle becomes a focused internet portal—perfect for distraction-free web reading, reference, or apps, all from your e-reader.
