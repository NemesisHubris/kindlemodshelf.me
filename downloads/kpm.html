KPM – Kindle Package Manager

KPM is a package manager for jailbroken Kindles, providing a simple way to install, update, and manage Kindle mods, apps, and utilities. No more downloading and copying files by hand—KPM can handle it all directly from your device.
What Does KPM Do?
  • Browse and install Kindle mods and utilities with a few taps
  • Update existing tools automatically when new versions are released
  • Remove or manage installed packages easily
  • Works with a growing library of Kindle apps and scripts
Requirements
  • A jailbroken Kindle
  • KTerm terminal emulator (for command-line usage)
  • KUAL (Kindle Unified Application Launcher) for launching the manager
How to Get KPM
GitHub Project: github.com/gingrspacecadet/kpm
Official Wiki & Documentation: kpmwiki.vercel.app/home
Summary
KPM brings simple, on-device package management to Kindle for the first time. Quickly set up new mods, keep your tools updated, and try new Kindle homebrew projects—all without ever leaving your e-reader.
