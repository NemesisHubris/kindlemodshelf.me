KindleKraft – Minecraft Server for Kindle

KindleKraft lets you run a Minecraft multiplayer server directly on your Kindle! Turn your jailbroken Kindle (with the right firmware) into a LAN Minecraft server for versions 1.7.x and 1.8.x. It’s fun, experimental, and perfect for creative tinkering.
What Is It?
KindleKraft transforms your Kindle into a Minecraft LAN server. Host classic Minecraft (1.7.x–1.8.x) multiplayer sessions right from your e-reader—no PC required once it’s set up!
Requirements
  • Jailbroken Kindle with softfloat (SF) firmware
    (Firmware must be below 5.16.3. KindleKraft is SF-only unless you manually replace the Java install for hardfloat.)
  • KTerm terminal emulator installed
How to Install
  1. Download KindleKraft:
    Get from GitHub Releases
  2. Copy files:
    • Put the extension in your Kindle’s /extensions folder.
    • Put the archive in the root directory of your Kindle (do not extract).
  3. (Optional) Create a 256MB swapfile: For best server performance, use a swapfile of at least 256MB.
How to Run
  1. Open KindleKraft in KUAL and select ‘Install KindleKraft’.
    This may take a while—please be patient!
  2. Once installed, select ‘Launch’ in KUAL.
    This opens KTerm.
  3. In KTerm, start the server by typing:
    kindlekraft
  4. Use kindlekraft -s to start the Minecraft server.
    Use kindlekraft -i to display your Kindle’s private IP address.
Connect & Play
  • Join the server from any device on the same LAN using Minecraft client version 1.7.x or 1.8.x.
  • For best compatibility, use the latest client with ViaFabricPlus.
  • Enjoy your own Kindle-based Minecraft SMP!
Notes
  • Tested and fully working on Kindle 7 (KT2, BASIC). Partially working on Kindle Paperwhite 7th Gen (PW3) up to firmware 5.16.2.1.1.
  • On newer Kindles or firmwares, UI glitches may occur, but core server functionality should still work.
  • This is a fun, experimental project—expect quirks!
  • By running KindleKraft, you agree to the Minecraft EULA.
