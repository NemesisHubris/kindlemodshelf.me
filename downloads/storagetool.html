StorageTool – Manage Your Kindle’s Storage

StorageTool is a storage management utility for jailbroken Kindle e-readers. It helps you analyze, visualize, and clean up your ebook library, giving you full control over your device’s storage space.
What Is StorageTool?
StorageTool lets you quickly see how your Kindle’s storage is being used, identify large files and duplicates, and organize your books. Whether you have a massive collection or just want to keep your Kindle running smoothly, StorageTool makes it easy.
Features
  • Storage overview: See a summary of your Kindle’s used and free space.
  • Scan your books directory: Analyze file sizes and types in your documents folder.
  • Format breakdown: View storage usage by file type (MOBI, PDF, AZW3, etc).
  • Recent files: Find newly added or recently modified books.
  • Duplicate finder: Identify and review potential duplicate files.
  • Settings menu: Set your main books directory, show hidden files, enable debug mode, and more.
  • Auto-updater: Check for and install the latest version of StorageTool from within the app.
Requirements
  • A jailbroken Kindle e-reader
  • KUAL (Kindle Unified Application Launcher)
  • KTerm terminal emulator (needed for installation and some features)
  • (Optional but recommended) KOReader for better file handling
How to Install
One-Line Install (Recommended)
  1. Open KUAL and launch KTerm on your Kindle.
  2. Run this command:
    curl https://jpt.bio/StorageTool/install.sh | sh
  3. Restart KUAL. You’ll see StorageTool in your menu.
Direct Install
  1. Download StorageTool from the GitHub repository.
  2. Copy the storagetool folder to your Kindle’s /mnt/us/extensions/ directory.
  3. Set executable permissions for the scripts:
    chmod +x /mnt/us/extensions/storagetool/run.sh
    chmod +x /mnt/us/extensions/storagetool/bin/storagetool.sh
  4. Restart KUAL to load the new extension.
How to Use
  1. Open KUAL and select StorageTool from the menu.
  2. Explore options to view storage usage, scan for large files, check recent additions, or find duplicate books.
  3. Adjust your settings from the tool’s menu to match your preferences (choose directory, show hidden files, etc).
Troubleshooting
  • Extension not showing up?
    Double-check that you installed StorageTool to the right folder and restarted KUAL.
  • Getting “command not found” errors?
    Check that all scripts have executable permissions.
  • Still having issues?
    Reinstall StorageTool or verify your installation steps.
