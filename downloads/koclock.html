Clock Plugin (dtDisplay) for KOReader

dtDisplay is a simple plugin for KOReader that shows the current time and date in a fullscreen widget—perfect for turning your e-reader into an ambient desk or bedside clock while charging.
Main Features
  • Fullscreen time and date display
  • Designed for e-ink screens
  • Great for use while your Kindle or Kobo is charging
  • Instantly closes with a tap—no need to exit KOReader
How to Install
  1. Download the plugin:
    dtDisplay on GitHub
  2. Unzip or clone the repository.
  3. Copy the folder into your KOReader’s plugins directory:
    /mnt/us/.adds/koreader/plugins/ (or the equivalent on your device)
  4. Restart KOReader if it’s running.
How to Use
  1. Open KOReader
  2. Go to Plugins → More tools → Time & Day
  3. The fullscreen time and date widget will appear
  4. Tap anywhere on the screen to close the widget
Compatibility
  • Tested on: Kobo Clara 2E, KOReader 2023.03
  • Should work on any device running KOReader with plugin support
