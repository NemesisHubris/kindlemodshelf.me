SOX Media Player for Kindle

SOX Media Player lets you play WAV audio files—including music and audiobooks—on your Kindle with Bluetooth audio support. Download the version for your firmware: softfloat (SF) or hardfloat (HF). Version 3 (HF beta) supports WAV files only.
Download SOX Media Player
  • SOX Media Player v1 (SF only): View on MobileRead
  • SOX Media Player v2 (SF only): View on MobileRead
  • SOX Media Player v3 (HF Beta, WAV only): Download soxhf.zip
Features
  • Bluetooth audio support—play through wireless headphones or speakers
  • Simple, no-frills interface for audio playback
  • v3 (HF beta): Latest version; currently supports only .wav files (other formats not yet supported)
Requirements
  • For v1 and v2: Jailbroken Kindle running softfloat (SF) firmware
  • For v3 (beta): Jailbroken Kindle running hardfloat (HF) firmware
  • Bluetooth support on your Kindle
  • WAV audio files for playback (HF beta supports only .wav)
How to Install
  1. Download the correct version for your firmware (see above).
  2. Unzip the downloaded file on your computer.
  3. Follow the included README instructions to place the files in the correct directory locations on your Kindle.
  4. Eject and disconnect your Kindle.
How to Use
  1. Open KUAL on your Kindle.
  2. Select SOX Media Player from the KUAL menu and launch it.
  3. Connect your Bluetooth audio device.
  4. Start playing your WAV files! (HF beta can only play .wav files.)
Notes
  • Compatibility: Versions 1 and 2 work only on softfloat (SF) firmware and will not run on hardfloat (HF) devices. Version 3 (beta) is for hardfloat (HF) firmware only, and currently supports only .wav audio files.
  • For full details, updates, and support, see the MobileRead threads for each version.
