KOWeather Plugin for KOReader

KOWeather is a weather forecast plugin for KOReader.
See live weather right on your Kindle or other e-ink device before you read outside.
Main Features
  • Current weather conditions: See what’s happening outside now
  • 3-day forecast: View today’s forecast and the next two days
  • Hourly forecast: Check the forecast by hour for the current day
  • Choose your location: Set your forecast by postal code
  • Celsius & Fahrenheit: (Switching units is in progress)
How to Install
  1. Download the latest release: KOWeather Plugin on GitHub
  2. Unzip the downloaded file on your computer.
  3. Copy the weather.koplugin folder to your KOReader device:
    Place it in /mnt/us/.adds/koreader/plugins/ (Kindle or Kobo).
    (If that folder doesn’t exist, create it.)
  4. Restart KOReader if it’s running.
  5. Open KOReader and find Weather in your Plugins menu.
How to Use
  1. Open the Weather plugin in KOReader.
  2. Enter your postal code to set your forecast location.
  3. View current conditions, forecasts, and hourly details on your e-ink screen.
Compatibility
  • Tested on: Kobo Libra H2O
  • Should work on other e-ink devices running KOReader, including Kindle
Notes
  • Weather data is provided by WeatherAPI.
  • KOWeather is a simple, lo-fi tool designed for e-readers—perfect for a quick check before you read outside.
With KOWeather, you always know if it’s the right moment to grab your Kindle and enjoy a book under the open sky.
