KindleFetch – The Easiest Way to Download Books Directly to Your Kindle

KindleFetch is a simple tool that lets you download eBooks straight to your jailbroken Kindle. No computer, no cables, no hassle—just search and download books right from your device using the built-in terminal.
What Is KindleFetch?
KindleFetch is a free, open-source tool for Kindle e-readers. It allows you to search for and download books from sources like Anna’s Archive and Library Genesis, all on your Kindle. It’s fast, simple, and saves you from sideloading files.
Key Features of KindleFetch
  • Download eBooks directly on your Kindle without a PC
  • Easy-to-use, text-based interface
  • Supports multiple popular book sources (Anna’s Archive, Library Genesis, and more)
  • Works with most jailbroken Kindle models and firmware (HF and SF)
  • Community maintained with regular updates
What You Need Before You Start
  • A jailbroken Kindle e-reader
    (Need help? Visit kindlemodding.org/jailbreaking/WinterBreak/)
  • KTerm terminal emulator
    (Essential for running KindleFetch and most mods)
  • KUAL – Kindle Unified Application Launcher
  • (Optional, but recommended) KOReader for better file handling
How to Install KindleFetch
Automatic Installation (Recommended):
  1. Open KUAL and start kterm.
  2. Enter this command and press Enter:
    curl https://justrals.github.io/KindleFetch/install.sh | sh
  3. Wait until installation finishes and you see a success message.
  4. Type exit to close kterm.
  5. Find “KindleFetch” in KUAL and launch it.
Manual Installation:
  1. Download the latest release from: github.com/justrals/KindleFetch/releases
  2. Unzip the files and move them to the extensions folder on your Kindle.
  3. Find and launch “KindleFetch” in KUAL.
How to Use KindleFetch
  1. Open KUAL and launch KindleFetch.
  2. Use the search prompt to look for the book you want.
  3. Choose your download source (Anna’s Archive, Library Genesis, etc.).
  4. Select your book and follow the instructions to download.
  5. Your book will appear directly on your Kindle—ready to read.
Troubleshooting and Tips
  • No search results or errors?
    Try again later—some book sources can be temporarily down or blocked. Changing your DNS to Google (8.8.8.8) may help.
  • Getting a “sign in” or HTML file from Z-Library?
    This means you’ve hit the Z-Library download limit. Use Library Genesis if possible.
  • Input errors or nothing happens?
    Double-check your command for typos, and make sure you’re using the latest KindleFetch version.
  • Keyboard issues?
    Only the default Kindle keyboard is supported in most cases.
